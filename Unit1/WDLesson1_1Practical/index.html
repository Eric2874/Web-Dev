Web Technologies

HTML

the standard language used to create and structure content for web pages. It uses tags to define elements like paragraphs, headings, images, and links, giving web pages their fundamental structure. While HTML provides the skeleton and content organization, CSS handles the visual styling, and JavaScript adds dynamic, interactive features.

CSS

CSS stands for Cascading Style Sheets. CSS describes how HTML elements are to be displayed on screen, paper, or in other media. CSS saves a lot of work. It can control the layout of multiple web pages all at once. External stylesheets are stored in CSS files.

JavaScript

JavaScript is a scripting language used to create and control dynamic website content, i.e. anything that moves, refreshes, or otherwise changes on your screen without requiring you to manually reload a web page. Features like: animated graphics. photo slideshows.

A-frame

a-Frame is a web framework for building virtual reality (VR) experiences. A-Frame is based on top of HTML, making it simple to get started. But A-Frame is not just a 3D scene graph or a markup language; the core is a powerful entity-component framewor that provides a declarative, extensible, and composable structure to three.js.
